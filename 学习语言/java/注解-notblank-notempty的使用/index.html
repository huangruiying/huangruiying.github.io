注解: NotBlank NotEmpty的使用

博客 标签 类目 主页

@NotBlank @NotEmpty (javax.validation.constraints.NotBlank)

  1. 接口标明 org.springframework.validation.annotation.Validated 注解
  2. 要校验的对象标明 javax.validation.Valid 注解
  3. 要校验的对象属性标明 javax.validation.constraints.NotBlank 注解
  4. 示例：
public class Demo{
    // 接口操作
    @PostMapping("/demo")
    @Validated
    public Wrapper<Response> demo(@RequestBody @Valid Request request) throws Exception {
		    Response response = method(request);
        return WrapMapper.<Response>ok().result(response);
    }

    public class Request {
	    @ApiModelProperty("开始日期")
	    @NotBlank(message = "开始日期不能为空")
	    private String begDate;

	    @ApiModelProperty("结束日期")
	    @NotBlank(message = "结束日期不能为空")
	    private String endDate;
    }
}
