Hugo教程 - 1 - 安装与部署

Tue, Jun 20, 2023

安装brew

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装hugo

$ brew install hugo

验证hugo

which hugo
/usr/local/bin/hugo

ls -l $( which hugo )
lrwxr-xr-x  1 nenglianjituan  admin  31  6 15 17:04 /usr/local/bin/hugo -> ../Cellar/hugo/0.113.0/bin/hugo

hugo version
hugo v0.113.0+extended darwin/amd64 BuildDate=unknown

使用hugo

# 初始化一个项目(此时有结构，但是还不能使用)
hugo new site hugoblog
cd hugoblog
# 创建文章 (content/post)
hugo new post/document.md (真实目录创建在：content/post/document.md)
# 下载一款皮肤 （安装皮肤 & 启动服务）
cd hugoblog
git clone https://github.com/spf13/hyde.git themes\hyde
# 指定皮肤，启动服务
hugo server -D -t hyde --buildDrafts

扩展-皮肤下载地址

试看主题效果
1. 下载皮肤
2. 放入theme文件夹内
3. cd theme/aaaaa/exampleSite
4. hugo serve  --themesDir ../..

应用主题效果
cp -r themes/[主题名称]/exampleSite/* [站点根目录]
hugo server -D

部署在 GitHub Pages 上

# 在GitHub上创建一个Repository,命名为: huangruiying.github.io
# 在本地使用hugo，生成静态页面到public:
hugo --theme=hyde --baseURL="https://huangruiying.github.io/" --buildDrafts

样式缺失

打包时指定好网站地址即可。–baseURL=“https://huangruiying.github.io/"

部分文章打包失败

draft: true 时标识笔记是草稿，使用 hugo --theme=hyde --baseURL="https://huangruiying.github.io/" 会失效 需要加–buildDrafts参数
